# Windows Vulnerability Notes

This folder contains notes on Windows vulnerabilities, exploitation details, mitigations, detection ideas, and related research.

## Suggested note types

- vulnerability summaries
- CVE-specific notes
- exploit chain observations
- patch and mitigation notes
- defensive detection opportunities
- references to relevant research and advisories

## Suggested structure

- one note per CVE, topic, or research article
- include affected components, impact, and references
- capture both offensive understanding and defensive takeaways
