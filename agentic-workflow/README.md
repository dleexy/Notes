# Agentic Workflow Notes

This folder contains notes on agentic workflows, orchestration patterns, tooling, evaluation, prompts, and implementation ideas.

## Suggested note types

- workflow and orchestration patterns
- prompt design notes
- tool use strategies
- memory, planning, and delegation ideas
- evaluation methods and benchmarks
- implementation lessons from experiments

## Suggested structure

- organize by topic, experiment, or date
- prefer concise notes with examples
- record tradeoffs, failures, and design decisions
