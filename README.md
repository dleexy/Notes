# Notes

This repository is a central place for curated notes and research.

## Scope

The repo should contain notes in these areas:

- Reverse engineering malware articles and research
- Agentic workflow ideas, patterns, and experiments
- Windows vulnerabilities, analysis, and references

## Purpose

The goal of this repository is to keep important technical notes organized, easy to review, and useful for future research and implementation work.

## Suggested structure

A simple starting structure could be:

- `malware-re/` for reverse engineering and malware research notes
- `agentic-workflow/` for agent design, orchestration, and workflow notes
- `windows-vulns/` for Windows vulnerability notes and references

## Notes style

Prefer concise, high-signal notes that are:

- easy to skim
- source-linked where possible
- organized by topic or date when helpful
- focused on practical insights, techniques, and references
